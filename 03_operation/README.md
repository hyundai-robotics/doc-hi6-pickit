# 3. 플러그인 관련

해당 섹션에서는 pick-it 플러그인에 적용된 내용들을 다룹니다.

pick-it 프로세서와 연관된 자세한 내용은 페이지 별로 안내된 pick-it 공식문서 링크로 확인 가능합니다.
